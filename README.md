readme_content = """# 🇧🇷 Projeto Informativo: Luiz Inácio Lula da Silva

Este é um projeto web completo (HTML, CSS e JavaScript) desenvolvido para apresentar de forma clara, responsiva e interativa a biografia, trajetória política, mandatos presidenciais e conteúdos multimídia sobre o atual presidente do Brasil, Luiz Inácio Lula da Silva.

---

## 🎯 Objetivo do Projeto
O principal objetivo deste projeto é consolidar e apresentar informações históricas e biográficas sobre a figura pública de Lula. O site serve como um portal informativo acessível, utilizando boas práticas de desenvolvimento front-end para garantir uma navegação fluida, acessibilidade e design responsivo (adaptável a celulares, tablets e computadores).

## 🛠️ Tecnologias Utilizadas
O projeto foi construído utilizando as três tecnologias fundamentais da web moderna:

* **HTML5:** Estruturação semântica de todo o conteúdo da página (seções, artigos, listas e incorporação de mídia).
* **CSS3:** Estilização visual customizada, tipografia, layout em blocos e regras de responsividade para diferentes tamanhos de tela.
* **JavaScript (ES6+):** Camada de interatividade do site, responsável pela manipulação do DOM, controle de eventos de navegação e efeitos dinâmicos.

## 🚀 Funcionalidades
* **Navegação Suave (Smooth Scroll):** Links do menu que levam o usuário diretamente à seção desejada com um deslize suave.
* **Vídeo Responsivo:** Incorporação de player do YouTube que se ajusta automaticamente (mantendo a proporção 16:9) em qualquer dispositivo.
* **Design Fluido:** Layout centralizado e limpo, otimizado para leitura com paleta de cores institucional (tons de azul e cinza escuro).
* **Seções Estruturadas:** Divisão cronológica e temática para facilitar o consumo da informação (Biografia, Trajetória, Mandatos e Mídia).
